# Maldaner Wiesbaden

Maldaner ist eine Institution. Seit 1859 in Wiesbaden ansässig, verbindet das Haus Kaffeeerfahrung über Generationen mit dem Anspruch, auch im Specialty-Segment zu überzeugen. Selten trifft Geschichte so direkt auf Handwerk.

## Tradition trifft Specialty

Was Maldaner von vielen jüngeren Röstereien unterscheidet, ist die Tiefe des Wissens, das über Jahrzehnte aufgebaut wurde. Die Röstprofile spiegeln dieses Erfahrungswissen wider – sorgfältig entwickelt, ohne modische Experimente.
